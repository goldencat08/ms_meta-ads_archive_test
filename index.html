Google Sheets 연동 설정이 필요합니다

이 파일 하단의 CONFIG 섹션에 Google Sheets API 키와 스프레드시트 ID를 입력하면 팀 전체가 데이터를 공유할 수 있습니다.
설정 전까지는 이 브라우저에만 임시 저장됩니다.

📋 설정 가이드 보기 ↓
📝 새 실험 로그 기록
⚙️ Google Sheets 연동 설정 가이드
Step 1. Google Cloud Console에서 프로젝트 생성 → Google Sheets API 활성화
Step 2. 사용자 인증 정보 → API 키 생성 (HTTP 리퍼러 제한 권장)
Step 3. Google Sheets에서 새 스프레드시트 생성 → 공유를 "링크가 있는 모든 사용자 편집" 설정
Step 4. 스프레드시트 URL에서 ID 복사 (예: 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms)
Step 5. 아래 CONFIG에 API 키와 스프레드시트 ID 입력 후 저장

Step 6. GitHub 계정 생성 → 새 Repository 생성 (public) → 이 index.html 업로드
Step 7. Settings → Pages → main branch 선택 → Save
Step 8. https://[계정명].github.io/[저장소명] URL 팀원과 공유 ✅
파일 내 CONFIG 섹션 (index.html 하단 <script> 안):

const CONFIG = {
  API_KEY: 'AIzaSyD-SczGHLBgKlZw0dgqF2ck-XFgvQCti3A',
  SHEET_ID: '1AzuYGZhRQF8tHltlFriTOjzafwvRqbQM5OHUIK7F7ug',
  SHEET_NAME: 'logs'
}
